Comparison Tool

The report generator tool allows the student to upload a csv file of the runtimes. It then plots the relevant graphs for the student and asks the student a few questions which can be answered by having a look at the graphs. Once, the student has completed answering all the questions, he can generate a tex file of the report which can be downloaded, and the student can then convert it into a pdf file. This allows the instructors to have a report, which is uniform for all the students, answers all the questions and is to the point, for ease in evaluation.

Data Entry

This feature of the website allows you to learn the subject by looking at the readings and the explanations generated by already coded approaches. The student can also download the code for these approaches and run them on his/her system. The student can compare various approaches and he/she can also compare the machines for the same approach.
